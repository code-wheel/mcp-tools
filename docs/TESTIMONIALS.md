# MCP Tools Testimonials Template

Use this template to collect short, specific testimonials from users.

## Consent + Usage

- Confirm the quote can be used publicly (website, Drupal.org, talks).
- Ask for name, role, and organization (or allow anonymous).
- Offer to share a draft back for approval.

## Quick Questions

1. What problem did MCP Tools solve for you?
2. Which workflow or tools did you use?
3. How much time did it save or what outcome improved?
4. Would you recommend MCP Tools? Why?

## Suggested Short Format

"[Outcome] with MCP Tools. We used [tools/workflow], and it saved [time] or improved [result]." — [Name], [Role/Org]

## Where to Collect

- Drupal.org issue template or GitHub Discussions
- Private Google Form for quick submissions
- Short outreach in Slack/Discord channels

## Example

"We rebuilt our blog structure in under 20 minutes using the blog template and workflows. It saved a full day of manual setup." — Alex, Drupal Lead
