# DrupalCon Talk Outline

## Working Title

"Drupal + MCP: Building Real Sites with AI Tools"

## Abstract

MCP Tools provides curated, Drupal-native capabilities for AI assistants. This talk covers real workflows (site health, content migration, and site building), the security model behind MCP Tools, and a live demo that builds a blog in minutes.

## Session Format

- 30 min session + 10 min Q&A (primary)
- 60 min workshop variant (optional)

## Key Points

- Why curated tools beat raw CRUD APIs
- MCP resources, prompts, and gateway mode
- Security: scopes, policy engine, audit logging
- Live demo: apply template, create content, run audit

## Learning Objectives

- Understand how MCP Tools fits into real Drupal workflows
- See a safe, repeatable AI-assisted site build
- Leave with a concrete demo script to try locally

## Audience

- Drupal site builders
- Drupal developers
- AI/ML curious teams

## Demo Script (10 minutes)

1. Start MCP Tools STDIO server
2. List workflows and preview blog template
3. Apply template and create a sample article
4. Run a site audit and show snapshot output

## Agenda (30 minutes)

1. Problem framing + MCP overview (5 min)
2. Drupal-native tool design + security model (7 min)
3. Live demo walkthrough (10 min)
4. Lessons learned + roadmap (5 min)
5. Q&A (3 min)

## Call to Action

- Install `mcp_tools`
- Try the demo playbook
- Contribute a use case or integration
